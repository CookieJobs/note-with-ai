## 目标
- 未完全展示的正文尾部采用淡入渐隐效果，视觉更优雅。
- 当内容被截断时，在正文区域显式显示【展开】按钮，展开后变为【收起】。
- 取消【编辑关键词】按钮，改为“点击任一标签即进入该关键词的编辑态”。

## 现状与锚点
- 截断与展开逻辑：`frontend/src/app/notes/page.tsx:237-269` 计算 `canExpand`，`wrapperRef` 控制 `maxHeight`；按钮：`frontend/src/app/notes/page.tsx:406-414`。
- 正文样式：`frontend/src/app/notes/notes.module.scss:629-648, 661-666`。
- 关键词区：`frontend/src/app/notes/page.tsx:691-747`（显示标签与“编辑关键词”按钮）。

## 技术方案
### 正文淡入渐隐
1. 在正文包裹容器内新增一个“底部渐隐遮罩”元素，当 `!isEditingContent && !isExpanded && canExpand` 时显示。
2. 样式使用线性渐变从透明到背景色，覆盖正文底部约 64px，高度可随行高自适配。
3. 结合现有 `maxHeight` 高度限制，渐隐层不会影响布局，仅做视觉过渡。

### 展开/收起按钮重构
1. 将按钮改为“卡片正文区域右下角的小型胶囊按钮”。
2. 仅在 `canExpand && !isEditingContent` 时显示，文案根据 `isExpanded` 在【展开】/【收起】间切换。
3. 保持现有 `isExpanded` 和 `maxHeight` 逻辑，点击时触发同样的状态切换。

### 点击标签编辑关键词
1. 移除“编辑关键词”按钮：删除 `frontend/src/app/notes/page.tsx:443-447` 渲染；样式中保留标签样式。
2. 为每一个关键词标签添加 `onClick`，进入“单标签编辑态”：
   - 新增状态：`activeKeywordIndex: number | null`、`tagEditValue: string`、`isEditingKeywords: boolean`。
   - 点击第 N 个标签时：`isEditingKeywords=true`，`activeKeywordIndex=N`，`tagEditValue=keywords[N]`。
   - 渲染时：将该标签位置替换为一个小输入框（仅替换该标签，不影响其他标签）。
   - 保存逻辑：更新数组的第 N 项并调用现有 `saveKeywords([...])`；支持 Enter 保存，Esc 取消，失焦也保存。
3. 关键词为空时，显示灰色提示文案“点击添加关键词”，点击后进入单标签新增编辑（在末尾追加一个输入）。

### 无障碍与交互
- 标签的交互添加 `role="button"` 与 `tabIndex=0`，支持键盘 Enter 进入编辑。
- 展开/收起按钮维持可聚焦与键盘 Space/Enter 触发。
- 保持现有乐观并发控制与 409 冲突处理逻辑不变（沿用 `saveKeywords`）。

## 代码改动点
- `frontend/src/app/notes/page.tsx`
  - 正文区域：在 `noteContent` 内新增渐隐层元素的条件渲染（依赖 `canExpand && !isExpanded && !isEditingContent`）。
  - 展开按钮：位置从 `noteContent` 底部统一改为右下角，样式用新的类名。
  - 关键词交互：为标签添加点击与键盘进入编辑；新增单标签输入的渲染与状态管理；删除“编辑关键词”按钮。
- `frontend/src/app/notes/notes.module.scss`
  - 新增 `.fadeOverlay` 渐隐层样式（线性渐变，绝对定位于正文底部）。
  - 新增 `.expandPill` 胶囊按钮样式（定位右下角，hover/active 态）。
  - 新增 `.keywordEditInput` 样式（小型输入，紧凑与标签一致的高度）。
  - 为 `.keyword` 增加可点击态（指针、轻微hover）。

## 兼容性与风险控制
- 保持 `isExpanded`/`canExpand` 计算与现有多行截断一致，不改动核心判定公式，避免布局回归风险。
- 单标签编辑依赖现有 `saveKeywords` 的 PATCH 接口与乐观并发处理；失败时恢复原值并提示错误。
- 样式只在笔记卡片内变更，不影响其他页面组件。

## 验证
- 正文截断卡片显示淡入效果；点击右下角【展开】后淡入层消失，按钮变为【收起】。
- 点击任一关键词标签进入编辑；修改后 Enter 保存、Esc 取消、失焦自动保存；关键词数组更新并写回后端。
- 空关键词时点击提示可新增一个关键词。

## 交付范围
- 仅涉及前端文件改动：`page.tsx` 与 `notes.module.scss`；不修改后端。
