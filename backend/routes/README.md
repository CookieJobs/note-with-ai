# 路由目录说明

本目录定义后端 REST 路由入口，负责将请求分发到对应控制器。

## 文件清单（名字 / 地位 / 功能）
- `auth.ts` — 路由 — 用户认证相关路由
- `cache.ts` — 路由 — 缓存相关路由（示例：清理/查看）
- `chat.ts` — 路由 — 聊天主流程路由
- `chatRelatedNotes.ts` — 路由 — 聊天相关笔记推荐路由
- `health.ts` — 路由 — 健康检查
- `notes.ts` — 路由 — 笔记 CRUD 与查询
- `performance.ts` — 路由 — 性能统计与监控接口
- `recommend.ts` — 路由 — 推荐内容接口
