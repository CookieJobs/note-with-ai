# 组件目录说明

本目录包含复用的 UI 组件与图标，服务于多页面与交互场景。

## 文件清单（名字 / 地位 / 功能）
- `CareAssistantPanel.tsx` — 组件 — AI 关怀助手面板
- `ChatHistoryPanel.tsx` — 组件 — 聊天历史列表
- `ChatInputArea.tsx` — 组件 — 聊天输入区
- `ChatMainContent.tsx` — 组件 — 聊天主内容区
- `ChatMessage.tsx` — 组件 — 单条消息气泡
- `ChatMessage.module.scss` — 样式 — 消息气泡样式
- `DeleteConfirmModal.tsx` — 组件 — 删除确认对话框
- `GlobalKeybindings.tsx` — 组件 — 全局快捷键绑定
- `MobileMenuButton.tsx` — 组件 — 移动端菜单按钮
- `MobileMenuButton.module.scss` — 样式 — 移动端菜单样式
- `NoteCard.tsx` — 组件 — 笔记卡片
- `NoteCard.module.scss` — 样式 — 笔记卡片样式
- `RecommendationItem.tsx` — 组件 — 推荐项展示
- `RelatedNoteCard.tsx` — 组件 — 相关笔记卡片
- `RelatedNoteCard.module.scss` — 样式 — 相关笔记卡片样式
- `RelatedNotesContainer.tsx` — 组件 — 相关笔记容器
- `RelatedNotesContainer.module.scss` — 样式 — 相关笔记容器样式
- `TopNavigation.tsx` — 组件 — 顶部导航栏
- `TopNavigation.module.scss` — 样式 — 顶部导航样式
- `icons/` — 目录 — 图标组件
