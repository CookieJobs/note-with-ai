# 前端应用目录说明

本目录为 Next.js 前端应用，采用 App Router 架构，页面、组件、样式与工具分层组织。

## 文件清单（名字 / 地位 / 功能）
- `src/app/` — 目录 — App Router 入口与各页面路由
- `src/components/` — 目录 — 复用的 UI 组件
- `src/hooks/` — 目录 — 业务与状态管理 Hooks
- `src/lib/` — 目录 — 前端库函数与 API 封装
- `src/styles/` — 目录 — 全局样式与变量
- `src/utils/` — 目录 — 通用工具方法
- `public/` — 目录 — 静态资源
- `next.config.js` — 配置 — Next.js 配置
